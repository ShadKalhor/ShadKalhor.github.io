# Shad Jalal Portfolio

A clean one-page portfolio website for Shad Jalal, Junior Backend Developer.

## How to use

1. Open `index.html` in your browser to preview the site.
2. Upload all files to your `ShadKalhor.github.io` GitHub repository.
3. Make sure GitHub Pages is enabled from repository settings.

## Files

- `index.html` — website structure and content
- `styles.css` — design and responsive layout
- `script.js` — mobile navigation and footer year
- `Shad-Jalal-CV.pdf` — downloadable CV
